# M5stack Experiments

## 起動メモ

USBでcores3をmacに繋ぐ。

UIFlow2.0にアクセス
https://uiflow2.m5stack.com/

繋いだデバイスを選択する。

最初に、画面左下のWebBurner（雷マークがある方）をクリックして、ファームウェアを書き込む。

（自分が試した時は、v2.0.5を書いた）

（多分、これを書かないと、自分で作ったプログラムを動かせない？）

すると、最初に買った時に動いていたプログラムとは別のものが動く。（左上にUIFlow2と書かれたもの）

それが表示された状態で、

画面右下の「Run」から、起動すると、自分が書いたプログラムが読み込まれて、

cores3に反映された！
